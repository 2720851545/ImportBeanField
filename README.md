# ImportBeanField

<!-- Plugin description -->
**ImportBeanField** 提供了快速添加Spring Bean到当前类中的功能。

源代码网址 [source_repo] 

- 🎥 [演示视频] 

## 快捷键 

### 打开选择Bean搜索框快捷键

Mac: `command + shift + i`

Windows: `ctrl + shift + o`

### 打开注入设置快捷键(仅在选中Bean搜索框时能打开)

Mac: `option + enter`

Windows: `alt + enter`
